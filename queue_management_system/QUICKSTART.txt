# QUICK START GUIDE

## Setup Instructions (Windows)

### Option 1: Using PowerShell (Recommended)

1. Open PowerShell or Command Prompt
2. Navigate to the project folder:
   ```
   cd C:\Users\Administrator\Desktop\queue\queue_management_system
   ```

3. Create virtual environment:
   ```
   python -m venv venv
   ```

4. Activate virtual environment:
   ```
   venv\Scripts\activate
   ```

5. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

6. Run migrations:
   ```
   python manage.py migrate
   ```

7. Create admin user:
   ```
   python manage.py createsuperuser
   ```
   - Username: admin
   - Email: admin@example.com
   - Password: (choose a password)

8. Start development server:
   ```
   python manage.py runserver
   ```

9. Open browser and go to:
   ```
   http://127.0.0.1:8000/
   ```

10. To access admin panel:
    ```
    http://127.0.0.1:8000/admin/
    ```

### Option 2: Using VS Code Terminal

1. Open VS Code integrated terminal (Ctrl + `)
2. Follow steps 2-10 from Option 1

## Initial Setup Steps

### 1. Create Admin User as Admin

After creating superuser:
- Go to http://127.0.0.1:8000/admin/
- Navigate to Users > User Profiles
- Edit your admin user
- Check "Is admin user"
- Save

### 2. Create Queue Services

Option A: Via Admin Panel
- Login to admin panel
- Go to Queue Services
- Click "Add Queue Service"
- Fill in details and save

Option B: Via Web Interface
- Login to the application
- Go to Admin Panel (only visible if you're admin user)
- Click "Create New Queue"
- Fill in and submit

### 3. Test the System

1. Create a test user:
   - Register new account
   - Login with new account

2. Book a token:
   - Click "Browse Queues"
   - Select a queue
   - Click "Book Token"
   - View your QR code

3. Track queue position:
   - Check real-time position updates

## Troubleshooting

### Issue: "python not found"
Solution: Make sure Python is installed and in PATH

### Issue: "No module named django"
Solution: Activate virtual environment and run: pip install -r requirements.txt

### Issue: Database error
Solution: Run: python manage.py migrate

### Issue: Static files not loading
Solution: Run: python manage.py collectstatic

### Issue: Port 8000 already in use
Solution: Run: python manage.py runserver 8001
